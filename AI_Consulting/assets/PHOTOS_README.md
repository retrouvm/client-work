# Data Center & Server Photos

## Where to Add Photos

Add your data center and server photos to this `assets/` folder with these names:

### Homepage Gallery:
- `datacenter-1.jpg` - Main data center photo
- `datacenter-2.jpg` - Server room interior
- `datacenter-3.jpg` - Server racks close-up
- `datacenter-4.jpg` - Data center overview

### About Page:
- `infrastructure-1.jpg` - Infrastructure photo
- `infrastructure-2.jpg` - Technology setup

### Services Page:
- `server-rack-1.jpg` - Server rack photo
- `server-rack-2.jpg` - Close-up server equipment
- `cloud-infrastructure.jpg` - Cloud infrastructure

## Image Recommendations:
- **Format:** JPG or PNG
- **Size:** 1200x800px or larger (will be automatically resized)
- **File size:** Keep under 500KB each for fast loading
- **Aspect ratio:** 3:2 or 16:9 works best

## Note:
If you use different filenames, update the HTML files to match your image names.
