# Header Image Setup

## Required Image

Please add Brenya Twumasi's close-up header photo to this folder and name it:
- `brenya-header.jpg` (or `.png`)

This image will appear in the header of all pages, showing her photo prominently when visitors first open the site.

## Image Recommendations

- **Format:** JPG or PNG
- **Size:** 400x400px or larger (will be displayed as circular 120-150px)
- **Aspect ratio:** Square (1:1) works best
- **File size:** Keep under 200KB for fast loading
- **Content:** Professional close-up headshot
